Quick Setup Checklist

  • Create Google Cloud project
  • Enable Google Sheets API
  • Enable Google Vision API
  • Create OAuth 2.0 credentials
  • Create API key for Vision API
  • Create Google Spreadsheet with proper structure
  • Update config.js with your credentials
  • Generate PWA icons
  • Test locally
  • Deploy to hosting service

Step 1: Google Cloud Console

1.1 Go to Google Cloud Console

1.2 Create a new project named "Shopping List"

1.3 Enable these APIs:

  • Google Sheets API
  • Google Cloud Vision API

1.4 Create OAuth 2.0 Client ID (Web application)

1.5 Create an API key (restrict to Vision API)

Step 2: Create Google Spreadsheet

2.1 Create a new spreadsheet named "Shopping List"

2.2 Create two sheets:

  • Shopping List - Headers: item, category, checked, addedBy, timestamp, order
  • Categories - Headers: category, storeOrder, useCount

2.3 Add default categories (Produce, Dairy, Meat, etc.)

2.4 Copy the Spreadsheet ID from the URL

Step 3: Configure the App

3.1 Open src/js/config.js

3.2 Replace these values:

  • clientId - Your OAuth 2.0 Client ID
  • apiKey - Your Google API Key
  • spreadsheetId - Your Spreadsheet ID
  • vision.apiKey - Your Vision API Key

Step 4: Generate Icons

Use one of these methods:

  • PWA Builder Image Generator
  • Favicon.io Emoji Generator (use üõí)

Save icons to src/assets/icons/ directory

Step 5: Test Locally

Run a local server:

python3 -m http.server 8000

Or: npx serve

Open: http://localhost:8000

Step 6: Deploy

Choose a free hosting service:

  • GitHub Pages - Push to GitHub and enable Pages
  • Netlify - Drag & drop or connect repo
  • Vercel - Import from GitHub

Don't forget to update your OAuth redirect URIs with the production URL!
